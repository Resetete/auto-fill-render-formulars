# KiLa Hildegarten — Formulare

Diese Web-App ermöglicht es Eltern und dem KiLa-Vorstand, wichtige KiLa-Formulare
am Computer auszufüllen und als PDF herunterzuladen, auszudrucken und zu unterschreiben.

Die App ist statisch aufgebaut und für GitHub Pages geeignet.

## Funktionen
- Übersichtsseite mit allen verfügbaren Formularen
- Formular ausfüllen → PDF generieren
- Unterstützung für ausfüllbare PDF-Formulare (AcroForms)
- Optional: Vorbefüllung über einen personalisierten Link (Token)

## Datenschutz
Die PDF-Erstellung erfolgt vollständig im Browser.
Es werden keine ausgefüllten Formulardaten durch diese App gespeichert.

## Technik (kurz)
- Statische HTML/CSS/JS App
- PDF-Erstellung im Browser
- Optional angebunden an eine externe Verwaltungssoftware über einen sicheren Vermittlungsdienst

## Deployment
Die App kann über GitHub Pages betrieben werden.

> Technische und interne Details (Konfiguration, Feldmapping, Wartung)
> sind in der internen Dokumentation (`ADMIN.md`) beschrieben.
